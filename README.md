## 🧭 What is aetherdev?

aetherdev is a local-first AI developer app for Windows. It helps you work with code, write files, and run helper tasks on your own computer.

It uses a local setup, so your work stays on your device. It is built for people who want a simple desktop tool that can help with everyday developer tasks without cloud setup.

## ✨ What you can do

- Ask for code help in plain language
- Generate project files and small app parts
- Work with a local AI model
- Keep your work on your computer
- Use a tool that can improve itself over time
- Run a self-hosted setup with open-source parts
- Use it for writing, editing, and code tasks

## 🖥️ What you need

aetherdev works best on a Windows PC with:

- Windows 10 or Windows 11
- 8 GB RAM or more
- A modern CPU
- 2 GB free disk space
- Internet access for the first download
- A local AI model tool if you want offline model use

If your PC is newer, it should run more smoothly. For larger models, more RAM helps.

## 🔧 First-time setup

After you open aetherdev for the first time:

1. Choose your local model option if the app asks.
2. Set the path to your model tool if needed.
3. Pick a workspace folder where your files will live.
4. Allow the app to scan or open that folder.
5. Test it with a short prompt like:
   - write a simple to-do app
   - create a README for my project
   - help me fix this code
6. Review the result before you save or use it

## 🧠 Local AI model setup

aetherdev is built for local use, so many users pair it with a local model runner such as Ollama.

Basic setup flow:

1. Install your local model tool.
2. Download a model that fits your PC.
3. Open aetherdev.
4. Link the model tool inside the app.
5. Run a test prompt.

Good starter models:

- Small code model for weak PCs
- Mid-size general model for most systems
- Code-focused model for better file help

If your PC has less RAM, use a smaller model.

## 📁 Suggested folder setup

Use a simple folder layout like this:

- `C:\AetherDev`
- `C:\AetherDev\Projects`
- `C:\AetherDev\Models`

This keeps your files easy to find. It also helps when you work on more than one project.

## 🛠️ Common tasks

### Create a project
Ask aetherdev to start a new app, script, or site. Give it a clear goal and the file type you want.

### Edit code
Open an existing file and ask for changes in plain words. For example:

- add a login screen
- fix the layout on mobile
- clean up this function

### Write docs
Use it to create README files, setup guides, and short notes for your project.

### Debug issues
Paste an error message and ask what it means. Then ask for a fix step by step.

## 🪟 Windows file tips

If Windows blocks the file:

1. Right-click the file
2. Open Properties
3. Choose Unblock if you see it
4. Select Apply
5. Open the file again

If the file is a zip archive:

1. Right-click the file
2. Choose Extract All
3. Pick a folder
4. Open the extracted app file

## ❓ Help with setup

If aetherdev does not start:

- Check that you downloaded the latest release
- Make sure the file finished downloading
- Try running it as an administrator
- Confirm that your model tool is installed
- Check that your workspace folder exists
- Restart the app after changes

If the app opens but does not respond:

- Close other heavy apps
- Try a smaller local model
- Make sure you picked the right model path
- Restart your PC and try again
